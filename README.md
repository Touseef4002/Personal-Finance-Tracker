# 💰 Personal Finance Tracker  

A **full-stack web application** that helps users efficiently manage their **income and expenses**, visualize spending patterns, and stay on top of their finances.  

---

## ✅ Features  
- 🔐 **User Authentication**  
  Secure login and signup system to protect user data.  
- ➕ **Add Transactions**  
  Add details like **amount, category, description, and date**.  
- 📦 **Data Persistence**  
  Stores all transactions in **MongoDB**.  
- 📊 **Dashboard**  
  Displays transactions in a **tabular format** with **sorting and filtering**.  
- 📈 **Data Visualization**  
  Interactive **charts** for income vs expenses and category-wise spending.  
- ✏ **Edit / Delete Transactions**  
  Update or remove transactions easily.  
- 🗂 **Categorization**  
  Organize transactions for better financial tracking.  
- 🔍 **Summary View**  
  Shows **Total Income, Total Expenses, and Balance**.  

---

## 🛠 Tech Stack  
- **Frontend**: React.js  
- **Backend**: Node.js + Express  
- **Database**: MongoDB  
- **Authentication**: JWT (JSON Web Token)  
- **Charts**: Chart.js or Recharts
